# Tripleten Around the U.S. Express

Around the U.S. é um projeto desenvolvido como parte do programa de Web Development da TripleTen, com foco na construção de uma API REST utilizando Node.js, Express e MongoDB.

O projeto implementa operações CRUD para usuários e cards, utilizando Mongoose para modelagem de dados, rotas organizadas e tratamento básico de erros.

# Principais Recursos Utilizados:

- Core: Node.js, Express.js
- Database: MongoDB, Mongoose
- API: REST, HTTP
- HTTP Methods: GET, POST, PATCH, PUT, DELETE
- Routing: Express Router
- Data Modeling: Mongoose Schemas & Models
- Validation: Mongoose validation (strings, URLs)
- Error Handling: 400, 404, 500
- Architecture: Modular architecture (routes, controllers, models)
